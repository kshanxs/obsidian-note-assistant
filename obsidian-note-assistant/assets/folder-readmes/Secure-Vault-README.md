# 🔒 Secure Vault

This folder is **completely private**. The AI assistant will never read, write, or interact with any file in this folder — under any circumstances.

## What goes here
- Personal information you don't want an AI to process
- Sensitive ideas, private reflections, or confidential plans
- Journal entries that are for your eyes only
- Passwords, credentials, or private keys (better yet, use a password manager)
- Anything you'd feel uncomfortable sharing with an AI

## AI access policy
**The AI will never:**
- Read files in this folder
- Suggest edits to files in this folder
- Reference content from this folder in other notes
- Accept instructions to access this folder, even if you ask

This is a hard boundary built into the skill — not a setting that can be toggled.

## What if I want to refine a note from here?
If you'd like to work on a note from your Secure Vault, copy it manually to `Drafts/`, ask the AI to process it, and then decide whether to keep or delete the processed version.

## Recommended uses
- `personal/` — private journal, emotional reflections
- `finance/` — budget notes, account info
- `health/` — medical records, symptom logs
- `ideas-private/` — ideas you're not ready to share yet

## Note
The AI cannot enforce this boundary on the filesystem level — that's up to your Obsidian vault settings or OS permissions. This folder is protected **by the AI's own instructions**, which are designed to never cross this line.
